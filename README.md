## 🚀 Hybrid System (npm & bun)

This project is optimized for speed and compatibility, supporting both **npm** and **bun** environments.

- 📦 **Lockfiles**: Both `package-lock.json` and `bun.lockb` are maintained. A `postinstall` script ensures they stay in sync.
- ⚡ **Runners**: Use `npm run <script>` or `bun <script>` interchangeably.
- 🛠️ **Helper Scripts**:
  - `npm run type-check` 🔍: Run TypeScript type checking.
  - `npm run clean` 🧹: Remove build artifacts and lockfiles.
  - `npm run reinstall` 🔄: Clean install via bun and sync npm lockfile.

## 🏁 Getting Started

First, spin up the development engine:

```bash
# Using Bun (Recommended)
bun dev

# Or using NPM
npm run dev
```

🌐 Open [http://localhost:3000](http://localhost:3000) to view the terminal.
